cryptocompare
#############

Python3 Wrapper to query cryptocurrency prices (and more) using the CryptoCompare_ API.


Installation
************

.. code:: bash

   sudo pip3 install python-cryptocompare

Usage
*****

Import
======

.. code:: python

   import python-cryptocompare

Usage
=======

Check unit tests to get a full list of examples.
